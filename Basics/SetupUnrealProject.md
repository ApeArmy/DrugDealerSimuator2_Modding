# Setting up your Unreal Engine project
First, make sure you downloaded Unreal Engine 5.3.2, only this exact version can be used for creating mods on DDS2.
1. Open Unreal Engine 5.3.2 from the epic games launcher.
2. Click on GAMES, then click on Blank.
3. make sure the target platform is Desktop and BLUEPRINT is selected.
4. Make sure the project name is exactly as follows: DrugDealerSimulator2
<img width="500" height="500" alt="UnrealEditor_JoznouvbTa" src="https://github.com/user-attachments/assets/448a361a-4105-4de1-aa40-82759214ead6" />

5. Click Create

# Configuring your Unreal Engine project
1. On the top left of unreal engine, click Edit > Project Settings
2. On the settings window that pops up, select the "Packaging" tab on the left
3. tick the checkbox for Generate Chunks.
4. Untick the checkbox for Share Material Shader Code
